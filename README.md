# Sales Data Analysis and Forecasting

## Overview
This project analyzes sales data from a Superstore dataset and builds a forecasting model.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Key Insights
- Technology category generates the highest profit.
- Sales show a consistent upward trend over the years.

## Machine Learning
- Built a Linear Regression model to predict future sales.

## Conclusion
The business is growing steadily, with strong performance in the Technology sector.
